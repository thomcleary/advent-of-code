# 🎄 👨‍💻 2019

[Advent of Code - 2019](https://adventofcode.com/2019)

## Setup

### 1. Install Gleam

<https://gleam.run/install/>

### 2. Save input

Save puzzle input as as `dayXX.txt` in `./priv/inputs`

- e.g. `./priv/inputs/day02.txt`

### 3. Run

```shell
$ gleam run 2
   Compiled in 0.04s
    Running aoc2019.main
 ▄▄▄▄▄▄▄▄▄▄
╭  Day 02  ────────────────────────────╮
│▀▀▀▀▀▀▀▀▀▀                            │
│ Part 1: 4023471                      │
│ Part 2: 8051                         │
╰─────────────── 18ms + 217ms = 235ms ─╯

$ gleam run 26
   Compiled in 0.03s
    Running aoc2019.main

usage: gleam run [{01|02|03|..|25}]
```
