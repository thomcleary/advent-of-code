# 🎄 👨‍💻 2019

[Advent of Code - 2019](https://adventofcode.com/2019)

## Setup

### 1. Install Gleam

<https://gleam.run/install/>

### 2. Save puzzle input to `./priv/inputs`

### 3. Run puzzle

```shell
$ gleam run
   Compiled in 0.03s
    Running aoc2019.main
usage: gleam run [day]

$ gleam run 2
   Compiled in 0.03s
    Running aoc2019.main
Part 1: 4023471
Part 2: 8051
```
